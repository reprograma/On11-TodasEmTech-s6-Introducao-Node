
1) Qual a relação entre os métodos HTTP e o CRUD?
R: CRUD são conceitos mais antigos que os verbos HTTP, que são similares, pois tem a mesma logica.

2) Comente, com exemplos, a diferença entre o PUT e o PATCH.
R: PUT é uma requisição completa dos dados, ja o PATCH é uma requisição parcial.

3) Assim como na aula, apresente os dados dos JSONs no console 
    - No colors-rgb.js apresente o nome da cor e o codigo RGB como no exemplo: "gainsboro - rgb(220, 220, 220, 1)" (feito)
    - No estados-cidade.js apresente o nome do Estado, a sigla e todas as cidadades, sem arrays aparentes no console (feito)
    - No filmes.js apresente titulo, plot, generos e lingua. Genero e lingua devem ser apresentados em arrays no console. (feito)

4) Defina o conceito de idempotência e como uma API pode ser idempotente
R: Idempotência é quando uma operação é feita sem que se altere o valor do resultado. Uma API pode ser idempotente quando fazemos uso dos "verbos http" que não façam alterações nos valores do JSON, por exemplo.
